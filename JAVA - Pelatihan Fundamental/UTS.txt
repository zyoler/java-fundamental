Dasar
• print, println, printf
Print itu mencetak output tanpa adanya format serta baris baru
printf itu sangat berguna jika ingin mencetak output dengan format
println itu menambahkan baris baru
• Scanner
Scanner itu adalah fungsi untuk menginputkan data/nilai menggunakan java.util.Scanner
• ForEach
ForEach fungsinya yaitu mencetak dengan mudah semua data array pada program tanpa perlu kita beri batas berhenti

Collection
• ArrayList
Array:List = Sekumpulan data yang berupa value
• HashSet
HashSet = Sekumpulan data tidak bisa ada data duplikat dan tidak urut
• HashMap
Hashmap = Sekumpulan data yang key dan value

OOP Abstraction
• Abstract
Abstract pada class : Membatasi bahwa class ini tidak boleh dibuat objek

Abstract pada method : Abtract itu ga punya isi
Non Abtract itu harus diisi

Abstract vs interface
Interface bisa untuk pewarisan lebih dari satu
Interface tidak bisa diberi access modifier(kecuali public)
Properti di interface selalu final, public, static dan harus memiliki value
Method di interface selalu abstract, public

OOP Polymorphism = Perubahan bentuk
• Method overloading
OVERLOADING yaitu memanggil fungsi sesuai dengan argumen yang kita masukan
• Method overriding
Override yaitu menimpa fungsi yang ada pada induknya
